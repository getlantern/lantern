linky (filter in module ngSanitize )

Description

Finds links in text input and turns them into html links. Supports http/https/ftp/mailto and plain email address links.

Usage

In HTML Template Binding

{{ linky_expression | linky }}

In JavaScript

$filter('linky')(text)

Parameters

  • text – {string} –

    Input text.

Returns

{string} –

Html-linkified text.
